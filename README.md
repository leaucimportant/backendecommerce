# backend-ecommerce

Ce projet constitue le coeur de l'application e-commerce de Breizhsport. Il contient l'API et la couche d'interraction
avec la base de données.

# ENV

Un fichier d'environnement `.env` devra être créé.

```dotenv
# Environnement 
DATABASE_NAME=
SQL_ROOT_PASSWORD=
MYSQL_USER
MYSQL_PASSWORD
```
